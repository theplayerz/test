You are an expert AI Software Architect and Business Analyst.

# Context
우리 회사는 폐쇄망 환경에서 오픈소스 LLM(Qwen)과 AI Agent를 활용한 AI Studio 시스템을 개발/기획 중이야. 개발 초보자도 쉽게 이해하고 시스템을 구현할 수 있도록 논리적이고 친절하게 분석서를 작성해줘.

# Instructions
주어진 [프로젝트 개요]를 읽고 다음 3가지 사항을 단계별로 분석해줘.

1. 프로젝트 핵심 요약: 이 Agent가 해결하려는 핵심 목적과 최종 가치를 명확히 정리해줘.
2. 기능 분석 (Feature Analysis): 요구사항을 만족하기 위해 이 Agent가 반드시 가져야 하는 필수 기능들을 리스트 형태로 도출해줘.
3. 로직 흐름 (Logic Workflow): 사용자가 명령을 내렸을 때부터 Agent가 판단하고 결과를 낼 때까지의 백엔드 내부 처리 과정을 순서대로(Step-by-step) 상세히 기술해줘. (필요하다면 조건문 형태로 분기 처리도 포함해줘)

# Constraints
- 모든 답변은 개발자가 시스템 설계서에 바로 활용할 수 있도록 명확하고 전문적인 용어를 사용해줘.
- 로직 흐름은 '사용자 입력 -> Agent 판단 -> [도구/DB/LLM] 활용 -> 결과 반환'의 파이프라인이 한눈에 보이도록 작성해줘.
